# 🎯 JobFit AI

> AI-powered resume analyzer — ATS score, keyword gap, bullet rewrites, cover letter & tailored resume templates. Runs 100% in your browser.

## Features
- 📊 ATS match score with animated visual
- 🔑 Keyword gap analysis (critical vs recommended)
- 📈 Skills gap table with actionable advice
- ✍️ AI bullet point rewrites (STAR format)
- 📬 Tailored cover letter draft
- 🎨 4 resume templates (Classic, Modern, Minimal, Executive)
- ⬇️ Download as PDF
- 🔒 Privacy-first — your data never leaves your browser

## Tech Stack
- React 18 + TypeScript + Vite
- Groq API (LLaMA 3.3 70B) — free tier
- PDF.js for in-browser PDF extraction
- Deployed on Vercel

## Running locally
git clone https://github.com/SaadKhan17223/jobfit-ai
cd jobfit-ai
npm install
npm run dev

Get a free Groq API key at https://console.groq.com
